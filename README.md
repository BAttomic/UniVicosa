## 🗂️ Organização do repositório

- Uma pasta por disciplina, no padrão **`CÓDIGO - Nome da Disciplina`**.
- Dentro de cada pasta, o material fica separado por tipo quando faz sentido:
  `Aulas/`, `Atividades/`, `Práticas/`, `Seminário/`, `Projetos/`.
- Disciplinas sem material digital têm um `README.md` com os dados da disciplina.
- O documento **[ESTRUTURACAO.md](ESTRUTURACAO.md)** registra a análise e o histórico
  da organização deste repositório.
