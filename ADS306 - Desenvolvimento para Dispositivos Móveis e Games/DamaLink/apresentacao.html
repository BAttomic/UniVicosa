Visão geral

Utilitário e jogo, lado a lado

O DamaLink combina em abas um consumidor de API de câmbio (AwesomeAPI) com uma partida de Damas brasileiras Humano × Bot. O foco técnico desta apresentação é o motor de regras das damas — onde mora a real inteligência do jogo — e a política de decisão do bot.

Motor de regras · 1/3

A regra de ouro: captura é obrigatória

A peça branca é obrigada a capturar a preta

Se existe qualquer captura no tabuleiro, os lances simples desaparecem da lista. A engine resolve isso numa única função: pergunta primeiro pelas capturas e só devolve movimentos normais quando não há nenhuma.

features/checkers/logic/checkersEngine.js

function getValidMoves(board, row, col) {
  const piece = board[row]?.[col];
  const captures = getCaptureMoves(board, row, col, piece);
  if (captures.length > 0) return captures;  // captura é OBRIGATÓRIA
  return getNormalMoves(board, row, col, piece);
}

Motor de regras · 2/3

A dama voadora: captura a distância

A dama (♛) varre a diagonal e pousa em qualquer casa após a vítima

Ao contrário da peça comum, a dama desliza pela diagonal a qualquer distância. A engine varre casas vazias até achar uma peça inimiga e então oferece todas as casas livres logo após ela como pouso válido.

checkersEngine.js — getKingCaptureMoves()

while (isWithinBounds(sr, sc) && !board[sr][sc]) { sr += dr; sc += dc; } // varre
const victim = board[sr][sc];
if (victim && victim.player !== piece.player) {
  let lr = sr + dr, lc = sc + dc;
  while (isWithinBounds(lr, lc) && !board[lr][lc]) {  // pousos possíveis
    captures.push({ row: lr, col: lc, capturedRow: sr, capturedCol: sc });
    lr += dr; lc += dc;
  }
}

Motor de regras · 3/3

Captura encadeada & promoção

Após a 1ª captura, a mesma peça continua capturando em cadeia

Depois de uma captura, se a mesma peça ainda pode capturar, o turno não passa: ela é forçada a encadear. E ao alcançar a última linha, é promovida a dama.

⛓️ Regras cobertas pela engine

  • Captura encadeada obrigatória (multi-salto)
  • Promoção automática ao chegar na 8ª linha
  • Peça comum captura nas 4 diagonais; move só à frente
  • Detecção de fim de jogo: sem peças ou sem lances

A inteligência do adversário

O "cérebro" do bot

O bot não faz busca em árvore como um motor de xadrez — a inteligência do DamaLink está nas regras. Sobre elas, o bot aplica uma política de lances legais: coleta todos os movimentos permitidos (com as capturas já filtradas como obrigatórias pela engine) e sorteia um. Simples, mas sempre joga dentro das regras — inclusive respeitando a captura forçada.

features/checkers/screens/CheckersGameScreen.jsx

// a engine já separou: se há capturas, só capturas entram na lista
const list = captureMoves.length > 0 ? captureMoves : moves;

// política do bot: escolha uniforme entre os lances legais
const choice = list[Math.floor(Math.random() * list.length)];
applyMove(choice.from, choice.to, getNow());

Evolução natural do projeto: trocar o sorteio por uma heurística gulosa ou minimax — a engine já daria o suporte.
